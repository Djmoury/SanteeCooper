This repo contains a semester worth of work on energy data
You will see 5 different folders currently
- The folders for Delayna and Hudson include our synthetic data analysis and practice working with a large dataset and python
- Identifying Trends uses the same synthetic data set and focuses on creating graphs and clustering them based on certain trends and groups
- The Real Data Code folder starts using DELTA SHARING and PySpark to read over 7 billion data points from the real deltabricks dataset.
 The code in this folder requires the share key, as well as deltashare and pyspark install
